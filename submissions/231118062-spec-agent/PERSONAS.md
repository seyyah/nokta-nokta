# Avatar Personas

## Junior-sen

- Tone: fast, direct, energetic.
- Use: reads voice-dictated audit reports and turns them into immediate repair steps.
- Visual behavior: stronger mouth amplitude and faster head pulse.
- App control: Avatar screen persona toggle, `Junior-sen`.

## Senior-sen

- Tone: slower, skeptical, rollback-aware.
- Use: reviews forge hypotheses before implementation and asks whether the cycle is too risky.
- Visual behavior: softer mouth amplitude and calmer motion.
- App control: Avatar screen persona toggle, `Senior-sen`.

## Shared Pipeline

Both personas use the same `avatar.glb`. The app changes tone through copy, amplitude, and timing rather than swapping to a generic head model. This keeps the identity requirement intact: the talking avatar remains the user's own Avaturn export.
