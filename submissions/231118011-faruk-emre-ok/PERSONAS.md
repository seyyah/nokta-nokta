# PERSONAS

## Student Operator

Owns the microphone permission, Avaturn identity, APK build, and final demo recording. The operator speaks into the Capture screen and checks that voice bars and mouth motion respond without visible lag.

## Human Expert

Joins the Jitsi bridge when a forge cycle is stuck. The expert watches the shared app screen, asks for the failing evidence, and gives one narrow decision that can be converted into the next commit.

## Coding Agent

Reads audit reports, makes scoped code changes only under the submission folder, records COMMIT/ROLLBACK/STUCK cycles, and refuses to invent proof artifacts.
